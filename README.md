# Computer Checkout System

A desktop application built with JavaFX and PostgreSQL to manage student computer reservations efficiently. This project was developed as part of the Software Engineering (SWE 212) coursework.

## Features
* **Student Management:** Add, update, delete, and view student records.
* **Computer Inventory:** Manage available computers, including brand and model details.
* **Reservation System:** Assign computers to specific students and track checkout data.
* **Database Integration:** Persistent and relational data storage using PostgreSQL.
* **User Interface:** Clean and interactive GUI designed with JavaFX and FXML.

## Tech Stack
* **Language:** Java 17+
* **GUI Framework:** JavaFX
* **Database:** PostgreSQL
* **Build Tool:** Maven
